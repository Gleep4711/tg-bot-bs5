# Tg Bot Bsv5

This project serves as a bots manager that forwards messages from various bots to a manager chat, organized by topics.

## Overview

The Tg Bot Bsv5 project is designed to streamline communication by centralizing messages from multiple bots and relaying them to a dedicated manager chat. By categorizing messages into different topics, it helps manage information flow efficiently.

## Features

- **Message Forwarding**: Forwards messages from various bots to a manager chat.
- **Topic Organization**: Allows categorization of messages into different topics for better management.
- **Centralized Communication**: Streamlines communication by aggregating messages in one manager chat.

## Installation

To run this project locally or deploy it, follow these steps:

1. Clone the repository: `git clone https://github.com/Gleep4711/tg-bot-bs5.git`
2. Navigate to the project directory: `cd tg-bot-bs5`
3. Install dependencies: `pip install -r requirements.txt`
4. Configure settings as needed.

## Usage

To use the bots manager:

1. Start the application: `python -m bot`
2. Set up bots to send messages to the manager chat.
3. Organize messages by topics in the manager chat for effective management.

## Configuration

Adjust the settings in the configuration file `.env` to tailor the bots' behavior, chat topics, or any other necessary parameters.
